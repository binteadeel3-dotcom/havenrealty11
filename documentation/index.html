Documentation

Welcome to Haven Realty. This is a high-impact, professional Real Estate HTML5 template designed with Bootstrap 5. It features premium glassmorphism UI, smooth AOS animations, and Swiper.js integration.

Note to Buyers: This is a static HTML template. To make forms (Contact, Tour Booking, Newsletter) active, you must connect them to a backend service or PHP script.

Project Structure

Your project is organized logically for easy editing:

HAVEN-REALTY/ ├── assets/ | ├── css/ (animate.css, aos.css, responsive.css, style.css) | ├── fonts/ (Inter font family - WOFF2) | ├── images/ (Categorized: blog, hero, icons, logo, properties, team) | ├── js/ (aos.js, main.js, swiper-bundle.js) | ├── vendors/ (Licenses) | └── videos/ (property-video.mp4) ├── index.html ├── about.html ├── properties.html ├── property-details.html ├── blog.html ├── agents.html └── ... Home Page Sections

1. Hero Section

High-impact background with a Glassmorphism filter. Includes a specialized search filter with two dropdowns for property types and cities.

2. Featured Properties

Uses Swiper.js. Each card is meticulously designed to show pricing, location, and property highlights with a "View Details" call to action.

3. Why Choose Us

A 3-column grid featuring high-quality icons. Perfect for building trust with your users.

4. Footer & Newsletter

The footer includes a modern Newsletter signup and detailed contact information (Address, Timings, Socials).

About Us Page

  • Hero: Consistent with the site theme for professional branding.
  • Company Mission: Dedicated layout for your brand story.
  • Team Section: Well-designed cards displaying agent photos, roles, and social links.

Properties Listing (Search)

This page is the heart of the browsing experience:

  • Sidebar Filters: Pre-styled checkboxes and dropdowns for City, Property Type, Price Range, and Square Footage.
  • Property Grid: Cards featuring "For Sale" or "For Rent" badges, price, and bed/bath counts.
  • Pagination: Functional UI for navigating through multiple property listings.

Property Details (Single View)

The most detailed page in the template, including:

  • Image Gallery: Full-width Swiper slider for interior shots.
  • Schedule Tour: A clean form for lead generation.
  • Specifications: Dedicated table for Sqft, Year Built, and Amenities.
  • Video Section: Embedded support for property walkthroughs (assets/videos/property-video.mp4).
  • Reviews: Post and read review sections with star ratings.

Blog & News

Includes 6 beautifully designed blog cards on the main page. The Blog Details page features a robust sidebar with categories, recent posts, and a project gallery to keep users engaged.

Global Customization

Colors & Branding

The entire template's color scheme can be changed in one place. Open assets/css/style.css: 

:root {
  --primary-color: #6c757d;  /* Change this to your brand color */
  --dark-color: #111111;     /* Main background color */
  --light-color: #f8f9fa;    /* Secondary background */
}

Image Replacement Guide

To replace images, follow these exact paths in the assets/images/ folder:

Hero Backgrounds: hero/hero-background.png
About Us Image: hero/about-us.png
Home Property Cards: properties/home-page-properties/
Search Page Listing: properties/properties-page-properties/
Brand Icons: icons/ (trust.png, secure.png, etc.)
Agent/Team Photos: team/ (agent1.png, member1.png)

Animations (AOS)

We use AOS (Animate On Scroll) for all transitions. To change an animation, find the data-aos attribute in the HTML: 

<div data-aos="fade-up" data-aos-duration="1000"> ... </div>

Options include: fade-up, fade-down, zoom-in, and flip-left.
